VERGIFAC Logiciel de facturation

Vergifac permet de saisir, d'archiver et d'imprimer des documents de type facture.
Le document final est enregistré dans un fichier de format PDF universellement utilisé qui a l'avantage d'être imprimé tel quel par toutes les imprimantes et d'être facilement joint à un mail.
La présentation du document est complètement paramétrable.
Le programme inclut une gestion 'client ou tiers' ainsi qu'une gestion 'produits ou services' mémorisant les informations nécessaires à une facturation aisée.
Selon l'activité de l'utilisateur les lignes détails de la facture ont:
_ soit des montants hors taxes avec calculs finaux du total hors taxes, des taxes et bases correspondantes et du toutes taxes.
_ soit des montants toutes taxes avec calculs finaux du total toutes taxes, des taxes et bases correspondantes.

Usage de Vergifac

Le répertoire actif doit être celui où ce trouve le programme vergifac.
Ceci permet au programme de trouver sans problème:
  _-le ou les dossiers base de données  ( pour le test: dbtestfac.db)
Les répertoires associés:
  _doc-aide contient ce manuel d'aide. Ne pas toucher.
  _histo-fac contient les factures définitives que vous avez sauvées après l'impression.
   (Vous pouvez supprimer les factures de test)
   C'est dans ce répertoire qu'il faut aller chercher vos factures pour les (re)imprimer ou les joindre à un mail.
 _memo-doc  contient les saisies sauvées.
  Vous pouvez supprimer les fichiers tests.
 _misenpage contient les fichiers de commande de la mise en page.
  Conseil: conserver les fichiers de démonstration. En faire des copies et les renommer avant de les modifier pour coder votre propre mise en page.
 _outils. Ne pas toucher. Permet la création de dossier.


SAUVEGARDE DE VOS FICHIERS
Sauvegardez:
_votre dossier(s) vergifac, base de données xxx.db
_les fichiers du répertoire histo-fac.
_les fichiers du répertoires memo-doc que vous avez créés.
_les fichiers du répertoire misenpage que vous avez créés.
Copier ces fichiers dans un répertoire différent de 'vergifac'
En cas de nouvelle version de vergifac, il vous suffira de les recopier à leurs places.
